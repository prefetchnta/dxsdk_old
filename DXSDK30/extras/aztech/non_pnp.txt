Installation for Aztech Non-Plug-and-Play devices:

Audio Devices
=============
1. Use Control Panel, Add New Hardware
2. Answer "NO" for auto searching of hardware
3. Select "Sound, video and game controllers"
4. Click on "Have Disk"
5. Select driver A or whatever the directory that contains the driver set
6. Select Manufacturer as "Aztech Labs"
7. Select Model as "Aztech 2316/R Sound I/II Audio Device"

Joystick Device
===============
1. Repeat steps 1 to 6 as above
2. Select Model as "Aztech 2316/R Sound I/II Gameport Joystick"
3. When prompt for the driver, select the Windows 95 CD or Direct-X CD
