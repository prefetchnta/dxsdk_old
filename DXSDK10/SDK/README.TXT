This directory contains all SDK components.

To ensure compatability with all editors, all tabs have been replaced
with spaces in all source files.

BIN:
    ddtest.exe  Test app for debugging DirectDraw
    dstest.exe  Test app for debugging DirectSound
    Sample app executables
  
DEBUG:  
    contains debug versions of all system binaries + binaries in BIN
    directory
    
INC:
    contains include files for DirectDraw, DirectSound, and DirectPlay

LIB:
    contains library files for DirectDraw, DirectSound, and DirectPlay
    *.LIB : COFF libraries (used by Microsoft Visual C++ 2.0 for example)
    *.LBW : 32-bit OMF libraries (used by Watcom C/C++ 10.0 for example)

SAMPLES:
    Contains all sample code.   See readme.txt file in this directory for
    more details.
